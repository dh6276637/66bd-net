# 66必读 (66bd-net)

简洁、高效、无干扰的科技资讯阅读平台。

## 功能特点

- 聚合多个科技资讯源（36氪、钛媒体、少数派、IT之家等）
- 中英双语文章展示
- 简洁无广告的阅读体验
- 自动定时更新内容

## 技术栈

- Python Flask
- MySQL 数据库
- 响应式设计

## 部署

```bash
pip install -r requirements.txt
python app.py
```
